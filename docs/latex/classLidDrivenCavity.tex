{\footnotesize\ttfamily void Lid\+Driven\+Cavity\+::\+Advance (\begin{DoxyParamCaption}\item[{bool}]{verbose\+\_\+advance }\end{DoxyParamCaption})}



Integrates the Lid-\/\+Driven Cavity simulation over time in serial. This function iterates over time steps, advancing the simulation state at each step. 

The simulation proceeds for a total of {\ttfamily NSteps} time steps, where {\ttfamily NSteps} is determined based on the total simulation time ({\ttfamily T}) and the time step size ({\ttfamily dt}).

During each time step, the simulation state is advanced by calling the {\ttfamily Advance} function. Optionally, verbose output can be enabled for the first time step for detailed logging.
